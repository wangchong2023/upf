## 3. Governance Automation (Autopilot Logic)

The `mgr-agent-orchestrator.js` script automates agent triggering based on the current milestone in `.milestone`:

- **IF** `.milestone` matches `CDCP` -> Trigger `agent-pm` + `agent-se`.
- **IF** `.milestone` matches `TR1` or `TR2` -> Trigger `agent-architect` + `agent-pm`.
- **IF** `.milestone` matches `TR3` or `TR4` -> Trigger `agent-dev` + `agent-se`.
- **IF** `.milestone` matches `TR5` -> Trigger `agent-tester` + `agent-qa`.
- **IF** `.milestone` matches `TR6` -> Trigger `agent-maintainer` + `agent-pm`.
